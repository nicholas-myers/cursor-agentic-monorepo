## Common Mobile Frameworks

### Cross-Platform

- React Native (JavaScript/TypeScript): setup docs https://reactnative.dev/docs/getting-started
- Flutter (Dart): setup docs https://docs.flutter.dev/get-started
- Ionic + Capacitor (web tech + native bridge): setup docs https://ionicframework.com/docs/intro/installation
- .NET MAUI (C#): setup docs https://learn.microsoft.com/en-us/dotnet/maui/get-started
- NativeScript (JavaScript/TypeScript + native UI): setup docs https://nativescript.org/docs/quick-start
- Cordova (webview-based bridge; older): setup docs https://cordova.apache.org/docs/en/latest/guide/cli/

### Native UI (Platform-Specific)

- SwiftUI (iOS): tutorial/docs https://developer.apple.com/tutorials/swiftui
- UIKit (iOS): reference/docs https://developer.apple.com/documentation/uikit
- Jetpack Compose (Android): setup docs https://developer.android.com/jetpack/compose/setup
- Android Views (Android XML/views; classic): docs https://developer.android.com/develop/ui/views

### Legacy / Still Seen

- Xamarin (older .NET cross-platform): docs https://learn.microsoft.com/en-us/xamarin/
