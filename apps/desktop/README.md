## Common Desktop App Frameworks

### Web Tech + Native Shell

- Electron (JavaScript/TypeScript + Chromium + Node.js): setup docs https://www.electronjs.org/docs/latest/tutorial/quick-start
- Tauri (Rust backend + web frontend): setup docs https://tauri.app/start/
- Neutralinojs (lightweight webview-based desktop apps): setup docs https://neutralino.js.org/docs/getting-started/your-first-neutralinojs-app
- Wails (Go backend + web frontend): setup docs https://wails.io/docs/gettingstarted/installation

### Native UI Toolkits

- .NET MAUI (C#; desktop + mobile): setup docs https://learn.microsoft.com/en-us/dotnet/maui/get-started
- Avalonia (cross-platform .NET desktop UI): setup docs https://docs.avaloniaui.net/docs/get-started/install
- Qt (C++/Python with PySide/PyQt): setup docs https://doc.qt.io/qt-6/get-and-install-qt.html
- GTK (primarily Linux, also cross-platform): setup docs https://www.gtk.org/docs/installations/
- Flutter Desktop (Dart): setup docs https://docs.flutter.dev/platform-integration/desktop

### Native Platform-Specific

- SwiftUI/AppKit (macOS): setup docs https://developer.apple.com/documentation/swiftui
- WinUI 3 / Windows App SDK (Windows): setup docs https://learn.microsoft.com/windows/apps/windows-app-sdk/set-up-your-development-environment

### Desktop Game / 3D-Capable

- Unity (C#): setup docs https://docs.unity3d.com/Manual/GettingStartedInstallingHub.html
- Unreal Engine (C++/Blueprints): setup docs https://dev.epicgames.com/documentation/en-us/unreal-engine/install-unreal-engine
